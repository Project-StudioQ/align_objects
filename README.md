# Tools:Q Align Objects

[English](README.en.md)

## 概要

View 3D / Pose Mode

オブジェクト/ボーンの位置、向き、大きさを指定したオブジェクト/ボーンと合わせる。

## UI

- Target
  - 移動元のオブジェクト/アーマチュアを指定する
- Sub Target
  - ボーンを指定する。
- Position
  - 合わせる軸を指定する
- Rotation
  - 合わせる軸を指定する
- Scale
  - 合わせる軸を指定する
